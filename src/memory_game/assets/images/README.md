# Imagenes de las cartas

Coloca aqui tus imagenes (una por **pareja**). Formatos: `.png`, `.jpg`, `.jpeg`, `.bmp`, `.gif`.

- Las imagenes se usan en orden alfabetico por nombre de archivo.
  Conviene nombrarlas `01.png`, `02.png`, `03.png`, ...
- Se recomiendan imagenes cuadradas (se escalan automaticamente).
- Numero de parejas necesarias segun la dificultad:
  - Facil (4x4): **8** imagenes
  - Medio (4x6): **12** imagenes
  - Dificil (6x6): **18** imagenes

Si faltan imagenes, el juego completa con cartas de color numeradas, asi que
funciona aunque esta carpeta este vacia.
